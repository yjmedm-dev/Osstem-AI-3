# Osstem AI - 해외법인 재무제표 자동 검증 시스템

> 매월 해외 법인 10곳에서 받는 재무 엑셀 파일을, 사람이 일일이 확인하지 않아도 자동으로 검토해주는 도구입니다.

---

## 이 프로그램이 필요한 이유

지금까지는 해외 각 법인에서 재무 데이터를 엑셀로 제출하면, 담당자가 직접 눈으로 확인했습니다.
이 과정에서 시간이 많이 걸리고 실수가 발생하기 쉬웠습니다.

이 프로그램은 그 수작업을 자동화합니다.
- **10개 법인**의 파일을 한꺼번에 처리합니다.
- 오류가 있으면 자동으로 찾아서 알려줍니다.
- 분석 결과를 보고서로 정리해줍니다.

---

## 주요 기능

### 1. 데이터 수집 및 정리
- 법인별로 제출된 엑셀/CSV 파일을 읽어옵니다.
- 법인마다 다른 통화(달러, 유로 등)를 원화로 자동 환산합니다.
- 계정 코드, 법인 코드 등 기준 정보를 자동으로 연결합니다.

### 2. 오류 자동 검증
제출된 파일에 문제가 없는지 자동으로 확인합니다.

| 검사 항목 | 어떤 오류를 찾나요? |
|-----------|-------------------|
| 대차 균형 | 왼쪽(차변)과 오른쪽(대변) 합계가 다를 때 |
| 계정 코드 | 본사 계정 목록에 없는 낯선 코드가 들어왔을 때 |
| 필수 항목 누락 | 반드시 있어야 할 계정이 빠졌을 때 |
| 이상 변동 | 지난달 대비 50% 이상 갑자기 늘거나 줄었을 때 |
| 내부거래 불일치 | 그룹 내 법인끼리 주고받은 금액이 서로 맞지 않을 때 |

### 3. 재무 분석
- 법인별 손익 현황을 요약하고 그래프로 보여줍니다.
- 주요 재무 지표(유동비율, 부채비율, 영업이익률 등)를 자동 계산합니다.
- 지난 기간과 비교해 크게 달라진 항목의 원인을 분류합니다.
- 목표(예산) 대비 실제 실적을 비교합니다.

### 4. 보고서 자동 생성
- 법인별 검증 결과를 보고서로 만들어줍니다.
- 오류/경고 항목 목록과 조치 방법을 함께 제공합니다.
- 경영진 보고용 대시보드 데이터를 출력합니다.

## 처음 시작하는 방법 (설치 및 실행)

### 사전 확인
- Python 3.10 이상이 설치되어 있어야 합니다. ([Python 공식 사이트](https://www.python.org/downloads/)에서 무료로 받을 수 있습니다.)
- Anthropic API Key가 있어야 합니다. (AI 기능 사용 시)

### 설치 순서

**1단계: 프로젝트 파일 받기**
```bash
git clone https://github.com/yjmedm-dev/Osstem-AI-3.git
cd Osstem-AI-3
```
> Git이 없다면, 상단의 초록색 `Code` 버튼 → `Download ZIP`으로 직접 받아도 됩니다.

**2단계: 필요한 라이브러리 설치**
```bash
pip install -r requirements.txt
```
> 이 명령어 하나로 프로그램에 필요한 모든 도구가 자동으로 설치됩니다.

**3단계: 환경 설정 파일 만들기**
```bash
cp .env.example .env
```
> `.env` 파일을 열어서 아래 항목을 채워주세요.
```
ANTHROPIC_API_KEY=여기에_API_키를_입력하세요
```

**4단계: 실행하기**
```bash
python src/main.py --corp all --period 2025Q4
```
> `--corp all` : 전체 법인을 처리합니다. (특정 법인만 하려면 법인 코드 입력)
> `--period 2025Q4` : 분석할 기간을 지정합니다. (예: 2025년 4분기)

## 오류 검증 규칙 상세

| 규칙 | 검사 항목 | 심각도 | 설명 |
|------|-----------|--------|------|
| V001 | 대차 균형 | 오류 (Error) | 차변 합계와 대변 합계가 다르면 반드시 수정해야 합니다 |
| V002 | 계정 코드 | 오류 (Error) | 본사 계정 목록에 없는 코드가 있으면 확인이 필요합니다 |
| V003 | 필수 계정 누락 | 경고 (Warning) | 법인 유형에 따라 반드시 있어야 할 계정이 빠진 경우 |
| V004 | 이상 변동 | 경고 (Warning) | 지난 기간 대비 ±50% 이상 변동된 항목 |
| V005 | 내부거래 불일치 | 오류 (Error) | 그룹 내 법인 간 거래 금액이 서로 맞지 않는 경우 |

> **오류(Error)**: 반드시 수정 후 제출해야 합니다.
> **경고(Warning)**: 검토 후 이상 없으면 그대로 진행할 수 있습니다.
